hexo+github快速搭建博客

第一步

  • 下载node.js
  • 下载git
  • 安装时一直按next即可
  • win+r输入cmd，输入node -v,npm -v,git -v 如果都能出现版本号则安装成功

第二步

  • cmd窗口输入 npm install -g hexo-cli 等待下载完成，输入hexo -v出现版本号则说明安装成功

第三步

  • 在任意位置创建文件夹，命名随意。在文件夹中右键 git bash here
  • 然后输入 hexo init
  • 然后输入 hexo s 按住Ctrl+左键点击链接
  • 如果出现hexo的主题则成功

第四步

  • 创建github账号，如果无法连接，用IP地址连接，还不成功就百度吧。
  • 点击自己的头像，选择自己的仓库。
  • 创建新仓库，命名为 你的用户名.github.io
  • 点击头像，点击setting
  • 点击有SSH的字样
  • 选择新建SSH KEY
  • 任意位置右键打开 git窗口
  • 输入 git config --global user.name 你的用户名 然后回车
  • 输入 git config --global user.email 你创建github时用的邮箱 然后回车
  • 输入 ssh-keygen -t rsa -C "你创建github时用的邮箱" 然后一直回车
  • 访问c盘用户文件夹下的.ssh文件夹下的id_rsa.pub文件,复制里面的内容
  • 回到github中创建ssh key地方，粘贴内容，随意起名字,点击添加
  • 回到你创建的仓库,点击code，选择ssh，复制内容
  • 到你创建的文件夹下，右键打开git窗口，输入 npm install hexo-deployer-git --save
  • 打开文件夹下的_config.yml
  • 划到最下面，改成这样
  • deploy:
  type: git
  repo: 你在code那复制的内容
branch: main
  • 保存退出

第五步

  • 在你创建的文件夹下右键打开git窗口
  • 输入 hexo c,hexo g ,hexo d 如果让你输入yes,就输入yes
  • 等待资源上传到github的仓库，在浏览器输入 你的用户名.github.io

    出现这样声明hexo的主题部署成功了

最后

  • 然后你就可以在github上搜索你需要的主题，按照对方的文档进行操作就可以安装对方的主题了。有关写
    作可以访问hexo的官方文档或百度。
